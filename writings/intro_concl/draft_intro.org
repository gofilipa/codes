* introduction

** intro sketch
Digitality has been coopted. 

We start with a meme from the DHS account, and read how that meme
consolidates a variety of fears into a single image.

Reading memes as literary objects.

There is an interpretive mechanism underlying the meme, but there is
also a technical mechanism, which is based on networks and recommender
algorithms.

The rest of the introduction lays out the conceptual framework for the
book, introducing the five digital "constraints" that guide the work
in each of the chapters: digitization, iteration, containment,
abstraction, and approximation. As the introduction explains, each of
these concepts work in a reductive way upon language data, which
methdologies form queer studies and related fields will push back
against.
